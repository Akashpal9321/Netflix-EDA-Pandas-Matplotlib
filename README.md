# Netflix-EDA-Pandas-Matplotlib
# üé¨ Netflix EDA using Pandas & Matplotlib  

This project explores Netflix's dataset using **Exploratory Data Analysis (EDA)** with **Pandas** and **Matplotlib**.  
The main goal is to find insights about Netflix movies and TV shows based on their **type, release year, duration, rating, and country**.

---

()Project Overview  
Cleaned and preprocessed Netflix dataset (handled missing values).  
Analyzed distribution of content type (Movies vs TV Shows).  
Explored movie durations and their patterns.  
Visualized trends using **Matplotlib** bar charts and histograms.
